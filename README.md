# 🍪 Food Optimization with Gradient Descent

Aplicación de algoritmos de **gradient descent** para optimizar procesos reales en la industria alimentaria. Este proyecto demuestra cómo el aprendizaje automático puede reducir defectos, minimizar costos y maximizar la calidad en líneas de producción de alimentos.

## 🎯 Casos de uso

| Producto | Objetivo | Variables optimizadas |
|---|---|---|
| 🍪 Galletas | Minimizar defectos + costo de desperdicio | Temperatura horno, tiempo cocción, cantidad harina, humedad, velocidad banda |
| 🥤 Bebida energética | Maximizar aceptación de sabor | Gramos de azúcar, % jugo natural, mg de cafeína |
| 🧀 Quesos | Minimizar lotes rechazados | Variables de proceso de maduración |

## 📊 Resultados

- Curvas de aprendizaje mostrando convergencia del modelo
- Visualización de convergencia de variables hacia valores óptimos
- Reducción del error de 0.060 → 0.026 en línea de galletas

## 🛠️ Tecnologías utilizadas

- Python 3
- NumPy
- Matplotlib
- Google Colab

## 📁 Archivos
